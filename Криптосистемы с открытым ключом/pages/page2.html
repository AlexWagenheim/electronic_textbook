Задача. С помощью алгоритма RSA выполнить шифрование и дешифрование  сообщения «ВАЗА».

 

Решение:

Выберем для простоты небольшие простые числа (в отличие от тех больших, которые выбирают при реальном кодировании).

 

Пусть pi =2 ,gi = 11.

Тогда ri = 2 * 11 = 22, φ(ri) = (2-1)(11-1) = 10.

Выберем в качестве сi число, взаимно-простое с 10 (т.е. НОД (сi,, 10) = 1), например сi = 3.

Выберем di из сравнения (3 * di) º l(mod 10). Таким (минимальным) числом является d i =7, т.к. (3*7) º l(mod 10), 21º l(mod 10). Получили   открытый ключ {сi, ri} - {3, 22},

                                         закрытый ключ {di, ri} - {7, 22},

Получим сообщение:

Зашифруем сообщение с помощью открытого ключа {3, 22} для того, чтобы послать его абоненту А.

Используем преобразование: w' = wc (mod ri,) < ri.

w'1 =  (33)(mod22) = 27(mod22) = 5, w'2 = (l3)(mod22) = 1, w'3 = (83)(mod22) = 512(mod22) = 6. w'4 = (l3)(mod22) = 1.

 

Получили сообщение w' = (5, 1, 6, 1).

 

Абонент A расшифрует это сообщение (5, 1, 6, 1) с помощью закрытого ключа {7, 22}:

Используем преобразование: w² = (w¢)d (mod ri,). w²1 = (57)(mod 22) = 78125 (mod 22) = 3, w²2 = (17)(mod 22) = 1, w²3 = (67)(mod 22) = 279936 (mod 22) = 8, w²4 = (17)(mod 22) = 1,

 

Проверим результат из условия: w = w²

(3, 1, 8, 1) → ВАЗА
