Решение задачи 1

Введённые данные

4·x1 + 2·x2 + 5·x3 → max
x1 + 2·x2 + x3 ≤ 430
3·x1 + 2·x3 ≤ 460
x1 + 4·x2 ≤ 450

Ответ

x1 = 0, x2 = 100, x3 = 230, F = 1350

Введённые данные

x1 + 2·x2 + x3 ≤ 430
3·x1 + 2·x3 ≤ 460
x1 + 4·x2 ≤ 450

Ответ

б) Нужно увеличивать производительность 2-го станка, это подтверждают расчёты:

 

Введённые данные

4·x1 + 2·x2 + 5·x3 → max
x1 + 2·x2 + x3 ≤ 430
3·x1 + 2·x3 ≤ 860
x1 + 4·x2 ≤ 450

Ответ

x1 = 0, x2 = 0, x3 = 430, F = 2150

 

 

в)

Расчёты демонстрируют, что производство изделия А всё ещё не выгодно

Введённые данные

6·x1 + 2·x2 + 5·x3 → max
x1 + 2·x2 + x3 ≤ 430
3·x1 + 2·x3 ≤ 460
x1 + 4·x2 ≤ 450

Ответ

x1 = 0, x2 = 100, x3 = 230, F = 1350

 

г) Если увеличить производительность 2-го станка на 200 минут, то получим:

Введённые данные

4·x1 + 2·x2 + 5·x3 → max
x1 + 2·x2 + x3 ≤ 430
3·x1 + 2·x3 ≤ 660
x1 + 4·x2 ≤ 450

Ответ

x1 = 0, x2 = 50, x3 = 330, F = 1750

 

Вычтем затраты и получим чистую прибыль: 1750-250=1500

1500 > 1350, следовательно, Такое вложение выгодно.
